integration

Drop-In Compatible

Three endpoint formats. Zero migration friction.

OpenAI SDK

POST /api/v1/chat/completions

Drop-in replacement for the OpenAI Chat API

Anthropic SDK

POST /api/v1/messages

Drop-in replacement for the Anthropic Messages API

OpenAI Responses

POST /api/v1/responses

OpenAI Responses API compatible endpoint

# OpenAI SDK — zero code change required
from openai import OpenAI

client = OpenAI(
    base_url="https://api.jbridge.ai/api/v1",
    api_key="your-jbridge-key"
)

response = client.chat.completions.create(
    model="gpt-5.4",
    messages=[{"role": "user", "content": "Hello!"}],
    stream=True
)

# — or use cURL directly —

curl -X POST https://api.jbridge.ai/api/v1/chat/completions \
  -H "Content-Type: application/json" \
  -H "Authorization: Bearer $JBRIDGE_KEY" \
  -d '{
    "model": "claude-opus-4-6",
    "messages": [
      { "role": "user", "content": "What is JBridge?" }
    ],
    "stream": true
  }'
